### üìú If,else if,else Directive:

<details>
<summary>Click Here For View</summary>

```scss
@mixin changeColor($value){
  @if $value > 20 and $value < 30{
    color: green;
  }
  @else{
    color: red;
  }
}

@mixin setFontSize($unit){
  @if $unit == sm{
    font-size: 1rem;
  }
  @else if $unit == md{
    font-size: 1.5rem;
  }
  @else{
    font-size: 2rem;
  }
}

.box{
  @include changeColor(22);
  @include setFontSize(blank);
}
```
